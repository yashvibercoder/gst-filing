- flipkart_reader.py
  Reads Flipkart Excel file
  Extracts: B2C, HSN, Documents, ECO
  
- amazon_reader.py
  Reads 14 Amazon files
  Extracts: B2C, B2B, HSN, Credit Notes
  
- einvoice_reader.py
  Reads E-Invoice file (when provided)

  No manual file lists needed
  Pattern: *A3SZBDZ05A1P39* for Amazon, etc.

  Phase 4: Detects which states have data
  Creates only necessary state folders
  Zero empty files!
  
- hsn_cleaner.py
  Removes letters from HSN codes
  95069990aa → 95069990

  Generates: B2B template CSV files

  Generates: B2CS template CSV files

  Output: 64 state folders, 1 file each = 64 CSV files
  
- hsn_b2b_gen.py

- hsn_b2c_gen.py

- documents_gen.py

- creditnotes_gen.py
  Generates: Credit notes template CSV files
  Input: Amazon credit note sheets
  Output: 64 state folders, 1 file each = 64 CSV files
  
- eco_gen.py
  Generates: ECO template CSV files
  Input: Flipkart Section 3 + Amazon E-Com GSTIN
